This is SpinalMap

What is it?
A JavaScript library containing generic functionality commonly used in Backbone/Bootstrap/ESRI mapping applications.
It is packaged as a collection of AMD modules and distributed using Bower.
Why did we create it?
Reusability...we were building multiple Backbone applications that required the same (or similar) components.
So, we decided to write the code in a more generic fashion and distribute it using a package manager.
Who is it for?
JavaScript developers that want to use this particular technology stack.
It is not WAB so it does require programming knowledge.
How do I get it?
Just like any other JavaScript library...
bower install https://git.gisinc.com/scm/js/spinalmap.git#develop --save
How do I use it?
Once it's installed, reference the scripts and stylesheets in your page (after the JSAPI) and require modules

require(['spinalmap/map-view', 
		 'spinalmap/grid/grid-collection', 
		 'spinalmap/grid/grid-collection-view', 
		 'jquery'], function(MapView, GridCollection, GridCollectionView, $) {
	var mapView = new MapView({
		el: $('main'),
		collection: config.services,
		mapOptions: config.mapOptions
	});

	mapView.on('layers-added', function(layers) {		
		var gridCollection = new GridCollection(config.grids);
	    var gridCollectionView = new GridCollectionView({ 
	    	el: $('main'), 
	    	model: mapView.model.layers, 
	    	grids: config.grids 
	    });
	    gridView.render();
	});
	mapView.load();
});
